The Story behind?

It is a sad one - literally. Basically I built this theme using FreewayPro. Unfortunately Softpress as its manufacturer retired some days ago. This basically means that we need to find an alternative. To be honest: There isn't any. But Pinegrow came into my mind and I'm trying to give it a go!
